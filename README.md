# 📚 Storybooks

A modern web application for discovering, reading, and managing your favorite storybooks. Built with Express.js and MongoDB, Storybooks provides an intuitive platform for users to browse, share, and engage with a community of story enthusiasts.

## ✨ Features

- **📖 Browse & Discover** - Explore a diverse collection of storybooks with easy navigation
- **🔍 Smart Search** - Quickly find your favorite storybooks using powerful search functionality
- **⭐ Save Bookmarks** - Create a personalized collection of your favorite stories
- **💬 Community Reviews** - Read and write reviews, rate storybooks, and engage with the community
- **👤 User Authentication** - Secure login with Google OAuth 2.0 integration
- **💾 Persistent Storage** - Cloud-based data management with MongoDB

## 🛠 Tech Stack

- **Backend**: Node.js with Express.js
- **Frontend**: EJS and Handlebars templating engines
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: Passport.js with Google OAuth 2.0
- **Utilities**: 
  - Express Session for session management
  - Body Parser for request parsing
  - Method Override for HTTP method support
  - Date-fns for date handling

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v14 or higher)
- **npm** or **yarn**
- **MongoDB** (local or cloud-based like MongoDB Atlas)
- A Google OAuth 2.0 application (for authentication)
